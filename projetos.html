Lojinha Social

Plataforma web full stack desenvolvida para digitalizar processos de ONGs, em parceria com a Prefeitura do Recife e UFPE, com foco em impacto social e escalabilidade. Atuei como Product Owner e desenvolvedor, definindo requisitos, priorizando backlog e organizando entregas com Scrum e Kanban, mantendo taxa de entrega acima de 90% por sprint. O sistema utiliza Node.js, React, Docker e Prisma, com APIs REST, banco de dados PostgreSQL, integração com API externa e testes unitários com Jest para garantir estabilidade. A plataforma é utilizada por mais de 100 ONGs, reduzindo até 60% das tarefas manuais operacionais e foi premiada como Melhor Solução e Melhor Pitch no programa CInBora Impactar.

React Node.js Prisma ORM PostgreSQL Docker Jest

CINOVA SGMP

Aplicativo mobile para gestão de condomínios, focado em comunicação interna e organização de demandas dos moradores. Iniciei o projeto com protótipo funcional em C++ para validar regras de negócio, reduzindo ~35% do retrabalho na implementação final em React Native. Atuei como Tech Lead e Scrum Master, definindo stack, arquitetura e liderando um time de 6 desenvolvedores, promovendo boas práticas, testes unitários e entregas semanais com Scrum e Kanban. O projeto foi selecionado para testes com 50 moradores, com build funcional em APK e ambiente padronizado via Docker, reduzindo ~40% do tempo de setup. O backend está em evolução com estudos em C# e .NET.

React Native API REST Prisma ORM PostgreSQL Docker Jest
